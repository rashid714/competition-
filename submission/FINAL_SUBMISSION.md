Final Submission Package — FoodRescue

Contents
- `submission.csv` — Kaggle submission file (generated by `scripts/kaggle_run.py`).
- `assets/thumbnail.png` — Thumbnail image for the Kaggle listing.
- `assets/architecture.svg` — Architecture diagram for reviewers (optional).
- `notebooks/kaggle_capstone_notebook.ipynb` — Runnable notebook to reproduce the demo in Kaggle/Colab.
- `README_for_kaggle.md` — Quick reproduction instructions for reviewers.
- `KAGGLE_CODE_SNIPPET.md` — runnable code snippet to reproduce the headless run.
- `RELEASE_NOTES.md` — release notes describing the package contents.

How to verify locally
1) Create and activate the Python 3.10 venv, then install dependencies (see `README.md`).
2) Run:
```bash
python scripts/generate_demo_data.py
python scripts/kaggle_run.py --session daily_YYYYMMDD --out submission.csv
```
3) Confirm `submission.csv` exists and `assets/thumbnail.png` is present.

Notes
- No API keys included. To enable Gemini-powered messages, set `GEMINI_API_KEY` locally or via `foodrescue-agent/.env` (do NOT commit keys).
- This package is curated for Kaggle reviewers and includes a runnable notebook that demonstrates generating the artifacts.
